# Nereels

## Описание

Тестовое задание - вертикальный слайдер для просмотра коротких видео (аналог Reels / Tik-Tok).

## Реализация

Для ускорения разработки использовался `Vite` в качестве сборщика и `SCSS` для стилей. Всё это можно реализовать и без
данных инструментов, просто немного дольше. Никаких JS-фреймворков не использовалось, только нативный JS.

Были созданы 2 класса: 

- `NereelsItem` - отвечает за отображение и управление воспроизведением одного видео (остановить, вопроизвести, 
включить/выключить звук, поставить лайк и т.д.);
- `NereelsList` - отвечает за сам листинг и подгрузку видео при скролле. 

Для плавного переключения между видео используется `scroll-snap`, а для автовоспроизведения видео в области видимости
используется `IntersectionObserver`. 

Для оптимизации работы страницы список видео рендерится постепенно, по мере скролла. Количество видео, отображаемое
при открытии страницы и добавляемое при скролле регулируется параметром `itemsToRenderCount` (минимальное значение: `3`).

При открытии страницы видео по умолчанию воспроизводится без звука. Это необходимо, т.к. мобильные браузеры не позволяют
воспроизводить видео с звуком до тех пор, пока пользователь не совершит какое-то взаимодействие с ним. После того, как
пользователь включит звук в видео, все следующие видео будут воспроизвоидиться с звуком.

**Возможная проблема:** в мобильных браузерах после включения звука некоторые видео, которые позже "догружаются" по скроллу, иногда 
не запускаются автоматически (снова требуются действия от пользователя). Если тапнуть по такому видео, 
оно воспроизведется. К сожалению, такая проблема присутствует и в референсах.

## Запуск

1. Установите зависимости `npm install`
2. Сделайте production-сборку `npm run build`
3. Запустите приложение `npm run preview` (если на локальной машине)
4. Для возможности просмотра на мобильном устройстве добавьте параметр `--host` при запуске `npm run preview`, например: `npm run preview -- --host`
5. Также собранное решение можно посмотреть по [ссылке](https://nereels.eyampolskiy.ru)
